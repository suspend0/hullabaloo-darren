Some of My Code

I do program some. I've put some classes that I have written over the last few years online here.

These look like javadoc; if you want to see the source click on the class name, as such... 

public class AhoCorasickPhraseMatcher
extends java.lang.Object
implements PhraseMatcher

  • Things for text processing.
    AhoCorasickPhraseMatcher.java
    Implementation of the Aho-Chorasick phrase matching algorithm, which is completely Unicode safe. I use the java.text.Collator to do the matching and word boundary detection (which may not always be a space in certain languages).
  • Things to make SAX processing a little more manageable.
    XMLFilterChain.java
    An easy way to bind a bunch of XMLFilters together.
    DocumentSplitterFilter.java
    A ContentHandler that allows you to treat one long repeating document as a series of individual documents.
    ElementSelector.java
    A useful way to detect if we are inside a certain element during SAX processing. Frees your content handler from tracking document location.
    SaxElementFilter.java
    A ContentHandler that strips selected elements from the sax event stream.
    SaxCharacterConsolidationFilter.java
    Consolidates multiple consecutive character() events into a single event.
    SaxTagExpanderFilter.java
    Expands encoded XML into sax events. For example, a fragment like 
    <div>&lt;p&gt;hi&lt;/p&gt;</div>
    can generate 
    startElement("div");
startElement("p");
characters("hi");
endElement("p");
endElement("div");
    instead of the usual 
    startElement("div");
characters("<p>hi</p>");
endElement("div");
  • Things for XML
    XMLBuilder.java
    Convenience class that makes building XML documents a little easier. The new J2SE 6.0 has a XMLStreamWriter as part of the new StAX api which is similar (well, better).
