Cygwin is a pretty cool unix shell environment for those of us who need to use windows desktops.

Some Cygwin Scripts

I need to use a windows desktop but I don't have to like it. So I install a proper command line and tool set. Woohoo!

  • Convenience Things
    ssh-agent script
    Wrapper so I can have ssh-agent running and not have to type my pass phrase over and over and over again.
    Open Script
    I just find open a lot easier to type than cygstart. Plus this matches what's on Mac OS X, and everybody knows being like a Mac is cool! 
    $ cat /usr/local/bin/open 
/usr/bin/cygstart "$*"
    I like it b/c I can just "open ." and up pops the windows explorer at present working directory, but I suppose "open file.html" and "open file.pdf" is my most common usage.
